## 快速开始

```sh
npm install hexo-cli -g

npm run server
```

## 创建新文章

```sh
hexo new -P path "文章标题"
hexo new -P path "文章标题" "标签1,标签2"
hexo new -P path "文章标题" "标签1,标签2" "分类1,分类2"
hexo new -P path "文章标题" "标签1,标签2" "分类1,分类2" "作者"
```
