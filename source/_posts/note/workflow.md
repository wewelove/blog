## 飞龙工作流引擎

在国产软件和企业数字化不断推进的背景下，越来越多团队开始寻找轻量、可控、贴合中国审批习惯的工作流引擎方案。飞龙工作流引擎（FlowLong），正是这样一款为中国式审批场景量身打造的国产工作流引擎。

飞龙工作流引擎主打 JSON 格式流程模型，整体设计轻量却不失强大，支持组件化集成，并提供了简易直观的审批流程设计器。无论是请假、报销，还是复杂的多级审批流程，都可以通过拖拽的方式快速完成配置，体验上高度贴近飞书、钉钉等主流办公平台。

项目整体围绕 “中国式审批” 进行深度优化，流程表达直观、配置成本低，非常适合中小企业或对流程灵活性要求较高的业务场景。

https://gitee.com/aizuda/flowlong

## jsonflow-ui

在流程可视化与易用性方面，jsonflow-ui 提供了一套完整且成熟的解决方案。该项目采用 100% 纯国产技术栈，并非传统 BPM 方案，而是围绕 JSON 流程模型构建的全新工作流引擎。

其最大特点在于：

- 支持 任意节点加减，并可在流程图中实时回显
- 内置自动布局算法，流程结构清晰直观
- 前后端采用 纯 JSON 交互，格式简单、扩展成本低

项目同时提供 在线零代码流程设计能力，支持办公申请、审批流配置等常见场景，UI 风格高度仿钉钉审批流，操作门槛极低。前后端代码完全自研，逻辑清晰，非常适合进行二次开发和深度定制。

在多端支持方面，jsonflow-ui 实现了一套代码适配 PC、Pad、H5 以及移动端 App，真正做到一次设计，多端使用。

https://flow.pig4cloud.com

## smart-flow-design

smart-flow-design 是一款基于 Vue 与 ant-design-vue 构建的自定义工作流解决方案，重点面向钉钉审批模式和国内企业常见流程场景。

该项目内置在线流程设计器，支持完整的流程流转能力，包括：

- 流程办理、退回、撤回
- 自由流转、会签、并行与串行流程
- 服务任务与智能提交
- 转办、委托、作废流程等高级能力

同时，系统提供清晰的 流程跟踪图与状态展示，可直观查看任务历史、节点流转信息以及任务分配情况，适合对流程可视化与可追溯性要求较高的业务系统。

https://gitee.com/crowncloud/smart-flow-design
https://gitee.com/crowncloud/smart-flow-design-v3

## Warm-Flow

如果你觉得 Flowable、Activiti 这类传统工作流引擎过于复杂、学习和集成成本较高，那么 Warm-Flow 是一个非常值得关注的国产替代方案。

Warm-Flow 主打 简洁、轻量、易扩展，整个流程引擎仅使用 7 张核心数据表，代码结构清晰，非常适合快速上手和业务集成。 在功能层面，它几乎涵盖了企业审批的所有常见需求，包括：

- 通过、退回、任意跳转
- 转办、终止、委派
- 会签、票签、加减签
- 并行与互斥网关

流程设计器支持通过 Jar 包方式集成，无需大量代码改造即可快速接入。同时，流程引擎自带流程图能力，即便不集成设计器，也能直接使用流程可视化功能。

数据库方面，Warm-Flow 目前已支持 MySQL、Oracle、PostgreSQL，并计划持续扩展对更多数据库及国产数据库的支持。

http://warm-flow.cn/

## 总结

从 飞龙工作流引擎 到 jsonflow-ui、smart-flow-design，再到 Warm-Flow，可以看到国产工作流在轻量化、易集成、中国式审批体验方面已经走得越来越成熟。

如果你正在寻找一套 可控性强、学习成本低、审批体验友好 的工作流解决方案，这些项目都非常值得深入研究和尝试。
